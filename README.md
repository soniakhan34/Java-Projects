# Java-Projects
This repository contains a collection of beginner-friendly Java projects developed to practice core Java concepts such as object-oriented programming, loops, conditional statements, methods, and basic problem-solving.

Each project is placed in its own folder with clear and readable code, making it easy for beginners and students to understand and learn.

----Purpose-----
1.Strengthen Java fundamentals

2.Practice problem-solving and logical thinking

3.Build a beginner-friendly Java portfolio
